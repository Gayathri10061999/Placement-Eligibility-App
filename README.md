# Placement Eligibility Streamlit Application

##  Overview

The Placement Eligibility Streamlit Application is a data-driven web application built for the EdTech domain. It enables placement teams to filter and shortlist students based on customizable eligibility criteria such as programming performance, soft skills scores, and placement readiness metrics.

This project demonstrates the integration of **Streamlit, Python (OOP), SQL, SQLite, and Faker** to create an interactive analytics dashboard.

---

## Problem Statement

Design and implement a Streamlit application where users can input placement eligibility criteria. Based on these inputs, the application queries a relational database of student data and dynamically displays eligible candidates.

---

## Business Use Cases

- Placement Management – Shortlist students using customizable filters.
- Student Performance Tracking – Evaluate technical and soft skill readiness.
- Interactive Analytics – Generate actionable insights for placement teams.

---

## Tech Stack

- Frontend: Streamlit
- Backend: Python
- Database: SQLite (or MySQL)
- Data Generation: Faker
- Concepts: OOP, SQL, Relational Databases, Data Analytics
